CIVITAS Starter

Provides the `/civitas_seed` command to apply idempotent starter seeds (permissions, government organizations, ranks, and rank_permissions).

Usage:
- From console: `/civitas_seed`
- In-game: `/civitas_seed` if your identifier is listed in `civitas-admin/config.lua` or `civitas-starter/config.lua`.

Important: This resource does NOT run automatically on start. Run the command manually after installation.
